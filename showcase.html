ENGE 1216 Project -- Wind Turbine Calculations

This project utilizes MATLAB in order to calculate wind turbine power. This program took in the data collected from the prototyping team while testing it using the test rig. The data collected was put into a spreadsheet that was imported into MATLAB, that output two graphs -- wind speed vs. power and wind speed vs. power coefficient.
